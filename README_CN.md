**Flinteger A1** 是一个小巧的iOS应用。它能禁止App的安装和删除，隐藏App来更近一步的保护你的隐私，限制打开App。网络防护能在Safari浏览器里拦截成人网站的访问，拦截钓鱼欺诈网站，恶意网站，加密货币挖矿网站，并允许自定义要拦截的网站。
